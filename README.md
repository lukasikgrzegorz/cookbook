#  SimpleCookbook 
A
pplication that allows you to search for recipes from around the world.

https://lukasikgrzegorz.github.io/cookbook

## 🚀 Run Locally

Clone the project

```bash
  git clone https://github.com/lukasikgrzegorz/cookbook
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

## 📝 Tech Stack

React, CSS
